Overview

jQuery Text Highlighter is a jQuery plugin for highlighting text fragments in HTML documents. Highlights are realized by wrapping selected text in span tags.

Features

  • Highlighting text that span multiple tags.
  • Removing highlights.
  • Setting custom highlight color and text wrapper.
  • Callbacks for highlighting and removing highlights.

Compability

This plugin has been tested to work in following browsers:

  • Mozilla Firefox 3, 3.6, 4, 5, 6, 7
  • Internet Explorer 7, 8, 9
  • Google Chrome 14
  • Opera 11
  • Safari 5.1

Dependencies

  • jQuery 1.4 or higher.
  • Functioning of jQuery Text Highlighter rely on DOM Level 2 Range object but IE prior to version 9 doesn't support it, providing TextRange instead. This plugin takes advantage of rangy – a cross-browser range library which provides standards-based API for performing common DOM Range tasks. The plugin is distributed with ("pack" version) and without rangy library, but bear in mind that the latter doesn't work in older versions of IE.

Usage

  • To use jQuery Text Highlighter include jQuery and plugin source file inside the <head> tag of your HTML document: 
    <script type="text/javascript" src="/path/to/jquery.min.js"></script>
<script type="text/javascript" src="/path/to/jquery.textHighlighter.pack.js"></script>
  • To setup the plugin on DOM element with id="sandbox", add the following code: 
    <script type="text/javascript">
    $(document).ready(function() {
        $('#sandbox').textHighlighter();
    });
</script>
  • jQuery Text Highlighter follows jQuery methods calling convention: 
    $('#sandbox').textHighlighter('setColor', '#ff0000');

Configuration

Property Type Default Description
color string #ffff7b Highlight color. Must be valid CSS color.
highlightedClass string highlighted Class given to wrapper of highlighted text.
contextClass string highlighter-context Class given to element on which plugin is initialized.
onRemoveHighlight function null Function that is called right before highlight removing. Highlight is passed to the function as param. If function returns false, highlight won't be removed.
onBeforeHighlight function null Function that is called before text highlighting. Range object is passed to the function as param. If function returns false, text won't be highlighted.
onAfterHighlight function null Function that is called after text highlighting. Array of created and normalized highlights is passed as param.
$.fn.textHighlighter.defaults object Defines default values for properties.
$.fn.textHighlighter.createWrapper function Function that should return wrapper which is used to wrap selected text (i.e. highlight that text). Actual options defined in initialization are passed as param.

Methods

Name Description
destroy() Disables plugin leaving all highlights untouched.
setColor(color) Sets highlight color. Color must be valid CSS color.
getColor() Returns highlight color.
removeHighlights([element]) Removes all highlights within given element or within context element if no element given.
